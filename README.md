-----DESCRIPTION-----

JobAdvisor was born from the idea to simplify people working experience, giving them an idea of what are they going to do
in their future job position. 

It connects People with companies, giving them a way to expand their toughts about a certain company.


----WEBSITE FUNCTIONALITY----

Base Funcionality: Login,Register User or Company

Special Funcionality: Create review, add job, get review, delete job,delete review, explore, get company, get job


----KEY FEATURES-----
User: create or delete review, profile management, work history management and apply for a job

Company: create or delete Job placement, Employee verification, company profile 


-- Technology --

- HTML5 & CSS
- Javascript (Front End & Back End)
- MariaDB database Server based
- NodeJS + Express


#How to run the project:


**Prerequisites** 

+ Node.JS installed 
+ MariaDB installed and running 
+ HeidiSQL or Workbench




----Get Started----

1) Open any database manager between HeidiSQL or others
2) create a new database named 'jobreview'
3) import the dump sql file 'dump_database' provided in the 'Database' folder (ensure triggers and verifications are active)
4) Open a terminal in the project root folder
5) install dependencies  '''bash npm install
6) open BackEnd/database.js
7) start the server using --> 'node server.js' OR 'node BackEnd/server.js'

The result will be the following: 
http://localhost:3000
